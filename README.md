Project Overview:
This project is a comprehensive case study focused on ,Target's retail operations in Brazil, based on a dataset of 100,000 orders placed between 2016 and 2018.Using SQL, I performed data cleaning, transformation, and analysis to uncover actionable business insights.

Tools Used:
